# Managing Deployments Using Kubernetes Engine 🚀

## 📌 Project Overview

This project demonstrates practical deployment management techniques using Google Kubernetes Engine (GKE).

The lab focuses on:

- Creating a Kubernetes cluster
- Deploying containerized applications
- Scaling workloads
- Performing rolling updates
- Managing ReplicaSets
- Observing zero-downtime deployment behavior

This repository documents my hands-on implementation and understanding of Kubernetes deployment strategies.
